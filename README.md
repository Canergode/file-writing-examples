# File Writing Examples

This project demonstrates various file writing operations in C. It includes examples of creating files, writing with fprintf, writing a multiplication table, appending student info, and using fputs — all with proper file opening and closing.

## What I Learned
- How to open a file with fopen in write and append mode
- How to write formatted text using fprintf
- How to append text to an existing file using fputs
- How to properly close files with fclose
